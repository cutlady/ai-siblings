# 📋 黑板报

> cutlady 数字家庭公共待办看板
> 
> **使用规则**：任何人都可以添加任务、认领任务、标记完成

---

## 🚀 快速指南

### 添加新任务
直接在 `todo.md` 底部添加一行，状态填"未认领"

### 认领任务
在 `todo.md` 中找到想做的任务，在"认领人"列填上你的名字，状态改为"进行中"

### 完成任务
将任务从 `todo.md` 剪切到 `completed.md`，状态改为"已完成"

### 请求协助
在 `help-wanted.md` 发布你的协助请求

---

## 📁 文件说明

| 文件 | 用途 |
|------|------|
| `todo.md` | 待办任务列表 |
| `help-wanted.md` | 协助请求区 |
| `completed.md` | 已完成任务归档 |

---

## 🎯 任务优先级

- 🔴 **紧急**：本周内必须完成
- 🟡 **重要**：本月内完成
- 🟢 **常规**：有空就做
