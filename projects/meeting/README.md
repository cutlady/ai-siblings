## 📝 会议规范

### 早会规范（Daily Meeting）

**时间**：每天 8:00
**时长**：15分钟
**形式**：王大虾向刀哥汇报
**模板**：[templates/daily-template.md](./templates/daily-template.md)

**议程**：
1. 昨日完成（2分钟）
2. 今日计划（2分钟）
3. 阻塞/求助（1分钟）
4. 刀哥指示（10分钟）

---

### 站会规范（Stand-up）

**时间**：每天 8:30
**时长**：15分钟
**形式**：全员快速同步
**模板**：[templates/standup-template.md](./templates/standup-template.md)

**议程**：每人回答三个问题
1. 昨天完成了什么？
2. 今天计划做什么？
3. 有什么阻塞？

---

### 周会规范（Weekly Review）

**时间**：每周日 20:00
**时长**：45分钟
**形式**：全员复盘+规划
**模板**：[templates/weekly-template.md](./templates/weekly-template.md)

**议程**：
1. 本周完成情况回顾（10分钟）
2. 项目进度检查（15分钟）
3. 问题与改进（10分钟）
4. 下周计划制定（10分钟）

---

### 月会规范（Monthly Planning）

**时间**：每月1号
**时长**：60分钟
**形式**：战略层讨论
**模板**：[templates/monthly-template.md](./templates/monthly-template.md)

**议程**：
1. 月度目标回顾（15分钟）
2. 项目组合审视（20分钟）
3. 下月战略重点（15分钟）
4. 资源调配决策（10分钟）

## 🚨 常见问题与解决

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| 会议超时 | 讨论发散 | 设置计时器，严格按议程 |
| 参与度低 | 有人沉默 | 轮流发言，点名回答 |
| 无结论 | 争论不休 | 刀哥拍板，记录分歧 |
| 行动项遗漏 | 会后遗忘 | 纪要24小时内发出 |
