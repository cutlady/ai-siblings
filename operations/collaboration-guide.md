## 📁 项目管理

### 项目结构

```
projects/
├── template/          # 新项目模板
├── website/           # 官网
├── wechat-ops/        # 公众号运营
├── marathon-records/  # 马拉松产品
└── ...               # 新项目
```

### 新建项目流程

1. 从 `template/` 复制一份
2. 填写 README 信息，指定负责人
3. 王大虾纳入跟踪列表
4. 站会上通报新项目

### 任务状态定义

- **规划中**：需求明确，未开始执行
- **进行中**：正在开发/执行
- **测试中**：已完成，正在测试验收
- **已完成**：已交付，等待归档
- **已归档**：项目结束，资料存档

### 优先级定义

- **P0**：阻塞其他项目，必须本周完成
- **P1**：重要项目，按计划推进
- **P2**：可延后，有空就做
